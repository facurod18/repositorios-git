# Data Science Portfolio

Portfolio of applied data science projects focused on business problems, public policy, social data, and real-world decision-making.

I am currently pursuing a master's degree in data science and have an academic background in political science. This portfolio is designed to show my ability to work across the full data workflow: problem definition, data cleaning, exploratory analysis, statistical modeling, machine learning, visualization, interpretation, and communication of results.
