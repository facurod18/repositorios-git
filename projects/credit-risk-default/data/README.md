# Data

Raw data is not included in this repository because the dataset was provided for a university course and redistribution rights are not confirmed.

Place the local dataset in:

```text
projects/credit-risk-default/data/raw/
```

Suggested local filename:

```text
PROYECTO.xlsx
```

The current analysis script expects this Excel file locally. The `data/raw/` folder is ignored by Git, so the dataset can be used on this computer without being published.
