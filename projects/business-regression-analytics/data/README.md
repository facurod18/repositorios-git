# Data

Raw data is not included in this repository because the datasets were provided for a university course and redistribution rights are not confirmed.

Place the local files in:

```text
projects/business-regression-analytics/data/raw/ecommerce.xlsx
projects/business-regression-analytics/data/raw/Consumer.xlsx
```

The `data/raw/` folder is ignored by Git, so the datasets can be used locally without being published.
