# Data

Place the raw Excel files in:

```text
projects/uruguay-political-ad-efficiency/data/raw/
```

Expected local files:

```text
BD_internas.xlsx
anuncios_generales.xlsx
BD_paginas.xlsx
dolar.xlsx
```

Raw data is intentionally not committed to GitHub. The repository publishes the reproducible analysis code, generated figures, and summary outputs while keeping raw datasets local until redistribution rights are confirmed.
